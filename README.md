# HIJIRI-DATE-PICKER

### Introduction:

The Hijri Date Picker (Arabic: romanized: al-taqwm al-hijr), also known as the Islamic date picker in English, is a lunar calendar consisting of 12 lunar months in a year of 354 or 355 days.

### Features:
	Show Arabic years, weeks, and dates. 
	Year picker
	Month picker
	Date picker

### Overview:
 This Widget consist of (Arabic)Hijri date picker where Georgian date picker is a usual date picker that we are using now. Hijiri date picker is based on the moon which is different from the normal date picker.
